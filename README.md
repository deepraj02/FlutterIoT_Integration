## API Reference

#### POST Led Number from Flutter App

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `api_key` | `string` | **Required**. Your API key |
| `led`     | `1-6`    | **Required**. Your API key |
